Interface RecoveringCDOSessionConfiguration

All Superinterfaces:

All Known Subinterfaces:
FailoverCDOSessionConfiguration, ReconnectingCDOSessionConfiguration
public interface RecoveringCDOSessionConfiguration extends CDOSessionConfiguration
A session configuration that uses a heart beat protocol to detect network problems. Subtypes specify the exact behavior to recover from these problems.
