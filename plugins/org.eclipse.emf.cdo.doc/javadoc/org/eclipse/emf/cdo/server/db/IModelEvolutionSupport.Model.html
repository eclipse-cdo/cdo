Klasse IModelEvolutionSupport.Model

java.lang.Object
org.eclipse.emf.cdo.server.db.IModelEvolutionSupport.Model
Umschließende Schnittstelle:
IModelEvolutionSupport
public static final class IModelEvolutionSupport.Model extends Object
A model stored in the DB store.

Models are loaded via model loaders when model evolution is performed. They provide access to the stored and registered (if available) EPackages as well as metadata like the model ID, original type, and time stamp.
