public interface ResourceFilter extends PermissionFilter
A representation of the model object 'Resource Filter'.
Since:
4.3

The following features are supported:

  • Path
  • Pattern Style
  • Folders
  • Text Resources
  • Binary Resources
  • Model Resources
  • Model Objects
  • Include Parents
  • Include Root
