Interface CDOEditorInput

  • All Superinterfaces:
    IAdaptable, IEditorInput
    All Known Subinterfaces:

    public interface CDOEditorInput
extends IEditorInput
    A specialized IEditorInput for the CDO editor. A CDOEditorInput is associated with a CDOResource, accessed through a CDOView instance.

    For related operations, see CDOEditorUtil
