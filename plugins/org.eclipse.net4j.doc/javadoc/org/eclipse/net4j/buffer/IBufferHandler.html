        void handleBuffer(IBuffer buffer)
        Handles an IBuffer and optionally releases it. The implementor of this method takes over the ownership of the buffer. Care must be taken to properly release the buffer if the ownership is not explicitly passed to some further party.
        Parameters:
        buffer - The buffer to be handled and otionally released.
