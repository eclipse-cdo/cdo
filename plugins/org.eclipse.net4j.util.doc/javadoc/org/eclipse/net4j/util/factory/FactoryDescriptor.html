A factory that delegates to an extension registry contribution.

Example contribution:

       <factory
             class="org.eclipse.net4j.util.concurrent.TimerLifecycle$DaemonFactory"
             productGroup="org.eclipse.net4j.util.timers"
             type="daemon"/>
