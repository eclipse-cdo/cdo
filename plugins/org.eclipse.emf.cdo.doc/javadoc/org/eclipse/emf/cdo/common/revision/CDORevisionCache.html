Interface CDORevisionCache

All Superinterfaces:

public interface CDORevisionCache extends CDORevisionCacheAdder, org.eclipse.net4j.util.event.INotifier
Caches revisions and possibly evicts those that are no longer strongly referenced when free memory runs low.
Since:
4.0
Author:
Eike Stepper
No Implement
This interface is not intended to be implemented by clients.
