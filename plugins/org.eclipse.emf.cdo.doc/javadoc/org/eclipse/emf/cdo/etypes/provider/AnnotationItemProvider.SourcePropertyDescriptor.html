protected class AnnotationItemProvider.SourcePropertyDescriptor extends ItemPropertyDescriptor
A specialized property descriptor for the annotation source feature.

This implementation that is AnnotationItemProviderAdapterFactory-aware, using its assistant when available. It specializes property value wrapper creation to provide assistant-driven nested property descriptors.
