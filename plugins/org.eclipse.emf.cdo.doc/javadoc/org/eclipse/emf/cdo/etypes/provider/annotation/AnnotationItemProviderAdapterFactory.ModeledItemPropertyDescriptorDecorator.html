public static class AnnotationItemProviderAdapterFactory.ModeledItemPropertyDescriptorDecorator extends ItemPropertyDescriptorDecorator
An item property descriptor decorator implementation used by AnnotationItemProviderAdapterFactory.createPropertyDescriptorDecorator to create an assistant-aware decorator. It is reusable in overrides of AnnotationItemProviderAdapterFactory.createPropertyDescriptorDecorator.
